## 📖 About

**Gemini for UWP** is an unofficial Google Gemini client built for the **Universal Windows Platform (UWP)**.

It runs on a wide range of Windows devices, including:

* 💻 Windows 10 PCs
* 📱 Windows 10 Mobile
* 🎮 Xbox One / Xbox Series (Developer Mode)
* 🖥️ Surface Hub
* 📦 Other UWP-compatible devices

The application uses **Google AI Studio API Keys**, allowing you to chat with Google's Gemini models directly from a native UWP application.

## ✨ Features

* 🤖 Google Gemini integration
* 💬 Clean chat interface
* 📝 Markdown message rendering
* ⚡ Fast and lightweight
* 🔑 Google AI Studio API Key support
* 🪟 Native Windows 10 design

## 🛠️ Built With

* C#
* XAML
* Universal Windows Platform (UWP)
* Visual Studio 2015

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/yourusername/Gemini-UWP.git
```

2. Open the solution in **Visual Studio 2015**.

3. Build the project.

4. Launch the application and enter your **Google AI Studio API Key**.

This project is **not affiliated with Google** and is an unofficial community-made client.
